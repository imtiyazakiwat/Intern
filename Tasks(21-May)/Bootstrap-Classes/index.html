Important Bootstrap Classes

Quick reference guide to essential Bootstrap classes

Layout

  • .container - Responsive fixed width container
  • .container-fluid - Full width container, spanning the entire width of the viewport
  • .row - Horizontal group of columns
  • .col - Column within a row

Content

  • .text-center - Center align text
  • .text-left - Left align text
  • .text-right - Right align text
  • .lead - Make a paragraph stand out
  • .blockquote - Blockquote formatting

Utilities

  • .m-* - Margin utility classes
  • .p-* - Padding utility classes
  • .d-* - Display utility classes
  • .align-items-* - Align items utility classes
  • .justify-content-* - Justify content utility classes

Components

  • .btn - Button component
  • .btn-primary - Primary button style
  • .card - Card component
  • .navbar - Navbar component
  • .modal - Modal component
